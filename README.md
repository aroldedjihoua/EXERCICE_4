#  BankManager - Application Java

##  Description
Application Java simulant un système bancaire simple avec différents types de comptes.

##  Objectif
- Créer une classe mère `BankAccount`
- Créer des classes filles :
  - `CheckingAccount`
  - `SavingsAccount`
  - `COD`
- Utiliser l’héritage en Java
- Manipuler des attributs et méthodes

##  Structure du projet
- BankAccount (classe mère)
- CheckingAccount (compte courant)
- SavingsAccount (compte épargne)
- COD (dépôt à terme)
- Main (tests)

##  Concepts utilisés
- Héritage
- Encapsulation
- Getters / Setters
- Constructeurs

##  Fonctionnement
Le programme :
- crée plusieurs comptes
- affiche leurs informations (solde, limite, durée, etc.)
